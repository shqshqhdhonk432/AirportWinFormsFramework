# AirportWinFormsFramework

Лабораторная работа по ППП / C#.

Настольное приложение Windows Forms (.NET Framework), реализующее управление тарифами аэропорта:

- хранение тарифов (направление, базовая цена, стратегия ценообразования, скидка);
- добавление, редактирование и удаление тарифов;
- загрузка и сохранение тарифов в JSON-файл;
- сортировка по столбцам в таблице;
- поиск направления с максимальной итоговой ценой.

Основные классы:
- `Tariff` — модель тарифа;
- `Airport` — коллекция тарифов и бизнес-логика;
- `FileService` — загрузка/сохранение JSON;
- `MainForm` — главное окно с таблицей (DataGridView);
- `TariffForm` — окно добавления/редактирования тарифа.
